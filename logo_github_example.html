Ejemplo: cómo referenciar el logo subido a GitHub (RAW)

Para que la etiqueta <img> use la imagen que subas al repo en la carpeta imagenes/, forma la URL "raw" así:

https://raw.githubusercontent.com/<TU-USUARIO>/<TU-REPO>/<RAMA>/imagenes/synapselogo.png

Reemplaza <TU-USUARIO>, <TU-REPO> y <RAMA> (por ejemplo main o master).

Ejemplo (sustituye por tus valores):

https://raw.githubusercontent.com/miusuario/mirepo/main/imagenes/synapselogo.png

Vista previa (si ya subiste la imagen):

Si ves la imagen —¡perfecto!—. Si no, revisa que la ruta sea exactamente la misma (GitHub diferencia mayúsculas) y que la imagen exista en la rama indicada.
